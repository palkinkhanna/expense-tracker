# Smart Expense Tracker

A Python based expense tracking application that helps users record expenses, categorize spending, and view monthly summaries.

## Features
- Add expenses with category and date
- Store data using SQLite
- View all expenses
- Monthly spending analysis
- Total expense calculation

## Tech Stack
- Python
- SQLite
- Pandas

## How to Run
1. Clone the repository
2. Install dependencies  
   pip install pandas
3. Run the app  
   python app.py
